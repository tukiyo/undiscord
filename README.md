Discordで自分の投稿を削除するスクリプト

F12キーを押してコンソールに貼り付け。

```
次に行う操作（文字化けしていても場所は同じです）
画面が見えるようになったら、以下の設定を行ってください。

AUTHOR ID の横にある me ボタンを押す（自分のIDが入ります）。

SERVER ID の横にある current ボタンを押す（現在のサーバーIDが入ります）。

CHANNEL ID の横にある current ボタンを押す（現在のチャンネルIDが入ります）。

Advanced settings（詳細設定）を開き、Delete delay を 1500 以上に設定します。

上部のメニューにある Delete ボタン（文字化けして 笆ｶ・・Delete みたいになっている赤いボタン）を押して削除開始です。
```

```
ステップ1：トークン（合鍵）を見つける手順
開発者ツールを開く

Discordの画面で F12 キーを押して、あの「英語の画面」を出します。

Network（ネットワーク）タブを開く

パネルの上の方にあるタブから 「Network」 を選んでクリックしてください。

（もし見つからない場合、「>>」のような記号の中に隠れていることがあります）

何か操作をする

Discordの画面で、適当なチャンネルをクリックしたり、誰かのアイコンをクリックしたりしてください。

すると、Networkタブにズラズラと通信記録が表示されます。

「science」を探す

Networkタブの左上にある「Filter（フィルター）」欄に science と入力してください。

science という名前の通信が見つかるはずです。それをクリックします。

Authorization（トークン）をコピーする

右側に出てきた情報の「Headers（ヘッダー）」タブを見ます。

下にスクロールしていくと、authorization: という項目があります。

その横にある、めちゃくちゃ長い英数字の文字列（例: MTA0... や Nz... で始まる文字）がトークンです。

これをコピーしてください。
```
